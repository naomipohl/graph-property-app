Draw your graph in the white box above.  Here's how to use it:

  • Add a vertex: double-click on the canvas
  • Add an edge: shift-drag on the canvas
  • Move something: drag it around
  • Delete something: click it and press the 'delete' key (use 'fn' + 'delete' on a Mac)
  • Type numeric subscript: put an underscore before the number (like "A_0")
  • Type Greek letter: put a backslash before it (like "\omega")
